## For Each Pair, Describe:

1. **Layout** — what's where? Toolbar position, sidebar, main content, input, status bar
2. **Colors** — background colors, text colors, button colors, border visibility
3. **Buttons** — how do they look? Flat? Gradient? Ghost? What color? Do they stand out appropriately?
4. **Spacing** — compact or spacious? Cramped or breathable?
5. **Overall vibe** — does it look sleek/modern or clunky/dated? Does it feel like Discord/Spotify dark theme or like a 2010 admin panel?
6. **Problems** — anything that looks broken, misaligned, too bright, too dim, unreadable, or ugly?

## Be Honest
Don't sugarcoat. If something looks bad, say so. If text is unreadable, say so. If buttons are invisible, say so. If spacing is off, say so.
